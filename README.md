# 🕹️ ZL_WEB.SYS | Personal Portfolio

A digital environment built with **React 19**, **Vite**, and **Tailwind CSS**. This project serves as my professional command center, featuring parallax effects, custom Aseprite animations, and a retro "Pink-Grid" UI system.

---

## 👾 System Overview
* **Status:** ONLINE 🟢
* **Framework:** React + Vite
* **Styling:** Tailwind CSS + Custom Pixel-Engine
* **Animations:** GSAP (Parallax) + Aseprite
* **Logic:** Responsive Web Components

---

## 🚀 Key Features
* **Parallax Layers:** Multi-depth scrolling backgrounds designed to mimic 16-bit era environments.
* **Custom Pixel Art:** Original assets created in **Aseprite**, including `gatorHeart.png` and `attack_on_titan_pixel_art.png`.
* **Integrated AI Bot:** A dedicated terminal interface for querying my professional experience via a Flask/Ollama backend.
* **Retro UI Elements:** Custom-styled scrollbars, "BigPixelMenu" navigation, and "AboutMe" modular boxes.

---

## 🛠️ Tech Stack
* **Frontend:** React 19, Vite, Tailwind CSS
* **Motion:** GSAP (GreenSock Animation Platform)
* **Graphics:** Aseprite, LibreSprite
* **Deployment:** GitHub Pages

## ⚙️ Boot Sequence

### 1. Clone the System
```bash
git clone [https://github.com/Zeitan0/my-portfolio.git](https://github.com/Zeitan0/my-portfolio.git)
cd my-portfolio
```

### 2. Install Dependencies
Install all required NPM packages
```bash
npm install
```
### 3. Launch Development Environment
Run the local Vite server
```bash
npm run dev
```

## 📂 Project Structure
* **/src/components** — Modular UI components (Parallax, AboutMeBox, ProjectGrid).

* **/src/assets** — Custom-drawn pixel art and animation exports.

* **/public** — Static files and the Zengtao_Liang_Resume.pdf source.

* **vite.config.js** — Configuration for GitHub Pages sub-directory deployment.
